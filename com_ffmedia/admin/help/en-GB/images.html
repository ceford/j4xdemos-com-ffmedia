Ffmedia Images

Folder Tree

The Folder Tree shows the first level of the folder tree with deeper levels for a selecteded branch of the tree. The selected folder is shown as an open folder icon.

Images List

No images are catalogued at installation. To get started, select the Folders menu item and then select Index All Folders from the Actions drop-down list in the Toolbar. Then come back here.

The list shows all of the images in the selected folder and all of its subfolders. Images can be filtered by any one or more of the following:

  • Search. Enter any text that may be in the file name, the alt description or the caption.
  • Status. May be Published (default), Trashed or Deleted.
  • Depth. May be Tree (default), meaning the selected folder and all its descendants, or Folder, meaning just the selected folder.
  • Extension. The list shows the extensions enabled in the Options configuration.

The thumbnail image is a link to a modal dialog which displays the image full-size or the width of the dialog, whichever is smaller.

The Actions list for each image allows selection of various types of tag to paste into a document, or to Trash, Restore or Delete an image, depending on the Status selection.

Toolbar Buttons

  • New. Opens a form to upload a new image.
  • Folders. A drop-down list allows creation of a new folder in the currently selected folder, or deletion of a folder if it is empty.
  • Records. Shows the number of filtered records.
  • Options. Opens the parameter settings form for this component.
  • Help. Opens a modal dialog containing this Help text.
