Ffmedia Files

Folder Tree

The Folder Tree shows the first level of the folder tree with deeper levels for a selecteded branch of the tree. The selected folder is shown as an open folder icon.

Files List

No files are catalogued at installation. To get started, select the Folders menu item and then select the Files option from the Media Type list. Then select Index All Folders from the Actions drop-down list in the Toolbar. Then come back here.

The list shows all of the files in the selected folder and all of its subfolders. Files can be filtered by any one or more of the following:

  • Search. Enter any text that may be in the file name, or the short or long descriptions.
  • Status. May be Published (default), Trashed or Deleted.
  • Depth. May be Tree (default), meaning the selected folder and all its descendants, or Folder, meaning just the selected folder.
  • Extension. The list shows the extensions enabled in the Options configuration.

The thumbnail image is a link to download the file, excepted when Deleted status is selected.

The Actions list for each file allows selection of various types of tag to paste into a document, or to Trash, Restore or Delete a file, depending on the Status selection.

Toolbar Buttons

  • New. Opens a form to upload a new file.
  • Folders. A drop-down list allows creation of a new folder in the currently selected folder, or deletion of a folder if it is empty.
  • Records. Shows the number of filtered records.
  • Options. Opens the parameter settings form for this component.
  • Help. Opens a modal dialog containing this Help text.
