Ffmedia Folders

This page shows the list of folders in use for storing either Images or Files. Use the Select Media Type drop-down list to switch. The list is displayed on the left of the screen, or near the top on narrow screens.

Select a Folder Tree item to perform one of the following actions:

Actions

Index Selected Folder
This action processes the files in the selected folder and inserts or updates a record in the database.
Index All Folders
This action processes all of the files in all of the folders and inserts or updates a record in the database. It may take quite a long time, perhaps several seconds for 1000 images. If some folders contain a lot of files it may appear to have stopped. Please be patient - there is no completion notice.
Index Selected Folder
This action processes all of the files in the selected folder and inserts or updates a record in the database. Use this option if indexing all folders runs into time-out trouble.
Hash All Folders
This action processes all of the files in the selected folder and updates the file hash field in the database record. It may take quite a long time. Perhaps several seconds for 1000 images. The hash field is not currently used but may be used in the future to identify identical files.
Hash Selected Folder
This action processes all of the files in the selected folder and updates the file hash field in the database record. Use this option if hashing all folders runs into time-out trouble.

Folders

New Folder
This action prompts for a new folder name and creates the folder. All characters except alphanumeric and minus (-) are replace by - (minus). You will be asked to confirm that the file name is acceptable. Please use minus characters to separate words.
Delete if Empty
This action deletes a folder only if it is empty.
