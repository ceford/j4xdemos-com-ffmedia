Ffmedia New or Edit File

New or Changed Files

For a new file all fields must be completed and a file selected. For an existing file you can change the metadata without changing the file. Or you can change the file without changing the metadata. Or you can change both at the same time.

File Tab

  • Browse. For a new file, this is the first item in the form. It allows you to select for upload any file in the list of allowed file types. When editing existing file metadata this item moves to the bottom of the list and becomes optional.
  • Short description. This is a few words to say what the file contains. It is used to create a human readable filename.
  • File name. This is constructed from the Short description text but only allows alphanumeric characters and the minus symbol, which is used as a word separator.
  • Long Description. This is used for keyword searching.
  • ID. This is the database record number set on record entry. It is useful for trouble shooting.

Toolbar Buttons

  • Save. Saves the File data and the file.
  • Save and Close. Saves and returns to the Files list page.
  • Cancel. Closes the form.
