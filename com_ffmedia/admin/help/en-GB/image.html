Photograph Sizes

It is best to prepare photographs for import - crop and resize for use on the web. Original photographs can be enormous and will take a long time to upload. Photographs larger than the limits set for this component or the hosting service will be rejected after upload. Photographs should be in jpg formt. Line art should be in png format.

New or Changed Images

For a new image all fields must be completed and an image file selected. For an existing image you can change the metadata without changing the file. Or you can change the file without changing the metadata. Or you can change both at the same time.

File Tab

  • Browse. For a new image, this is the first item in the form. It allows you to select for upload any image file in the list of allowed image types. When editing existing image metadata this item moves to the bottom of the form and becomes optional.
  • Alt attribute. This is a few words to say what the image contains. It is used to create alt text used by screen readers and a human readable filename good for SEO purposes.
  • File name. This is constructed from the Alt attribute text but only allows alphanumeric characters and the minus symbol, which is used as a word separator.
  • Image Caption. This is used to create caption tags and for keyword searching.
  • ID. This is the database record number set on record entry. It is useful for trouble shooting.

Toolbar Buttons

  • Save. Saves the File data and the file.
  • Save and Close. Saves and returns to the Files list page.
  • Cancel. Closes the form.
